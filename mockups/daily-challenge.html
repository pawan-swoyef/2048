Shared across every direction

• Rotation: game = (puzzleNumber − 1) mod 4 over kGames order — same date, same game for everyone, no backend.
• Per-day seed: each game builds a date-seeded board from the existing dailySeed(today).
• Goal & metric: 2048 → reach 512 / moves · Number Tap → tap 1–25 / time · Number Sort → sort all / moves · Magic Square → lines = 15 / time.
• Streak, resume, share: reuse the existing DailyStore keyed by puzzle number, plus the shared win/result card with a "next daily (game) in …" countdown.
• Directions differ only in chrome (how the header / today's-game / goal are presented) — the board and result card are the same building blocks.
